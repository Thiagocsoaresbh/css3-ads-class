# Desenvolvimento Responsivo com CSS

## Sobre a aula

Esta aula tem como objetivo introduzir os conceitos fundamentais de CSS e
Desenvolvimento Responsivo, abordando desde a função do CSS até a adaptação
de layouts para diferentes tamanhos de tela.

Todo o conteúdo foi pensado para iniciantes e pode ser executado apenas com:
- Navegador (Chrome, Edge ou Firefox)
- Visual Studio Code (opcional)

Não é necessário instalar frameworks ou bibliotecas.

---

## Objetivos de Aprendizagem

Ao final desta aula, o aluno será capaz de:

- Entender o papel do CSS na estilização de páginas HTML
- Diferenciar seletores de classe e ID
- Utilizar pseudo-classes como :hover
- Compreender o conceito de design responsivo
- Aplicar a abordagem Mobile-first
- Utilizar media queries para adaptar layouts
- Criar uma página simples responsiva

---

## Estrutura do Projeto

- Pasta `aula/`
  Contém o código base apresentado pelo professor.

- Pasta `exercicios/`
  Contém os enunciados dos exercícios que os alunos devem realizar
  modificando o código da aula.

---

## Como utilizar este repositório

1. Faça o download do projeto ou clone o repositório
2. Abra a pasta no Visual Studio Code (opcional)
3. Abra o arquivo `aula/index.html` no navegador
4. Siga as orientações do professor
5. Realize os exercícios descritos na pasta `exercicios/`
